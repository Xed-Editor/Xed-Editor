**Karbon** is a powerful text editor designed for developers, programmers, and power users on Android. It offers advanced features like syntax highlighting and robust customization, making it a perfect tool for text management on the go.

---
## Features
  - [x] File editing
  - [x] Open Multiple Projects
  - [x] Edit Multiple files
  - [x] Word Wrap
  - [x] Undo-Redo functionality
  - [x] Syntax Highlighting
  - [x] Search and Batch Replacement
  - [x] Integrated File Browser
  - [x] Create, Rename, Copy, and Delete Files/Folders
  - [x] Print text documents
  - [x] AMOLED Theme
  - [x] Keybindings
  - [x] Built-in Terminal
  - [x] Termux Interoperability
  - [x] Run Python, Node.js, etc.
