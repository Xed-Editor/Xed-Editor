## The Future of Coding, On Your Phone

Xed-Editor transforms your Android device into a **powerhouse development environment** — a full-featured code editor, terminal, and **AI coding assistant** all in one. Powered by a multi-agent AI architecture, it brings the same capabilities as desktop AI coding tools right to your pocket.

Whether you're fixing bugs on the go, prototyping ideas, or building full projects, Xed-Editor's AI agents **read, write, and edit your code autonomously** — just tell them what to do.

## Why Xed-Editor?

### 🤖 AI, Amplified
Choose from **5 AI engines** — a native built-in vibe-coding agent plus external agents (Gemini CLI, OpenCode, Codex CLI, Antigravity). Save agent profiles, configure per-project, and switch effortlessly. Your AI, your way.

### 🧠 Truly Autonomous
The Model Context Protocol (MCP) bridge gives AI agents **direct access to over 30 editor tools** — they can read, write, search, refactor, run commands, and even manage git — all without leaving the chat. The native vibe-coding agent runs entirely on-device with its own tool ecosystem.

### 🎨 Stunning, Native UI
Built with Jetpack Compose and Material 3, Xed-Editor feels right at home on Android. Smooth animations, adaptive theming, and an intuitive interface make coding a pleasure.

### 📝 Pro-Grade Editor
Powered by the **SoraX** engine with syntax highlighting, minimap, multi-cursor, code folding, bracket autocomplete, and a built-in color picker.

### 🔬 Code Intelligence
Built-in LSP servers for Python, TypeScript/JavaScript, HTML, CSS, JSON, XML, Markdown, Bash, and more — with real-time diagnostics, intelligent completion, go-to-definition, and hover docs.

### 🖥️ Full Terminal
A proper **Termux-based terminal emulator** with proot/Ubuntu support, session management, extra keys, and pre-installed launchers for every AI agent.

### 🔎 Search at Scale
Blazing-fast code and file search backed by **Room database indexing**, plus in-file find/replace and a command palette with 30+ actions.

### 🎨 Your Editor, Your Rules
Customize everything — Material 3 themes, separate fonts for editor/terminal/UI, custom keybinds, icon packs, and a plugin system for extending functionality.

### 🔒 Private by Design
**Zero tracking, zero telemetry, zero analytics.** No Firebase, no data collection. Fully open source under GPL v3 — your code stays yours.

## Getting Started

### Download

| Source | Link |
|:---|---|
| GitHub Releases | [Download Latest APK](https://github.com/algospider/Xed-Editor/releases) |
| Nightly Builds | [GitHub Actions (debug)](https://github.com/algospider/Xed-Editor/actions) |

### Quick Start

1. **Install** the APK on your Android device (Android 8.0+)
2. **Open** the AI Sheet from the toolbar
3. **Configure** your API key for any supported AI agent
4. **Start coding** with AI at your fingertips
